Bagaimana Cara Kerjanya?

Sistem ini menggunakan algoritma deteksi air melalui sensor kelembapan tanah/hujan. Arduino memproses input analog dari sensor untuk menggerakkan mekanisme jemuran.

Logika Sistem: Jika pembacaan sensor pada Pin A0 melebihi nilai 100, sistem mendeteksi adanya air/kelembapan tinggi, sehingga Servo pada Pin 4 akan bergerak ke posisi 90° (menutup). Jika nilai di bawah 100, servo kembali ke 0° (membuka).

Ada Air: Servo Menutup (90°)

Kering: Servo Membuka (0°)

Hardware Skeleton

5V / VCC Connection
GND / Ground Connection
Analog A0: Moisture Sensor
Digital 4: Servo Signal

Skema di atas menunjukkan koneksi wiring antara Arduino Uno, Sensor Kelembapan/Hujan, dan Motor Servo.

Source Code

v1.0.0 
#include <Servo.h>

Servo servo_4;

void setup()
{
  pinMode(A0, INPUT);
  Serial.begin(9600);
  servo_4.attach(4, 500, 2500);
}

void loop()
{
  int val = analogRead(A0);
  Serial.println(val);

  if (val > 100) {
    servo_4.write(90);
  }
  if (val < 100) {
    servo_4.write(0);
  }
  delay(10); 
}

Pastikan library Servo.h sudah terinstal di IDE Arduino Anda sebelum melakukan upload code.
